## Overview
โครงการนี้คือระบบบริหารจัดการร้านกาแฟและร้านอาหารที่ออกแบบมาเพื่อช่วย จัดการโต๊ะ ออเดอร์ การเงิน และพนักงาน ได้ในที่เดียว ช่วยให้เห็นสถานะโต๊ะและการทำงานแบบเรียลไทม์ เพื่อให้เจ้่าของธุรกิจและพนักงาน บริหารธุรกิจแและทำงานง่ายขึ้น

## Problem
- **ร้านกาแฟ/ร้านอาหาร**: ที่มีโต๊ะจำนวนมากทำให้เกิด ความสับสน เช่นโต๊ะไหนว่าง? โต๊ะไหนสั่งแล้ว?
- **พนักงาน**: เสียเวลาเดินเช็กสถานะโต๊ะ ดูรายการอาหาร ดูยอดบิลด้วยตนเอง ทำให้บริการล่าช้า
- **การปิดบิล/จัดการเงิน**: ซับซ้อน และในบางครั้งเงินหาย จัดการเงินได้ยาก
- **เจ้าของกิจการ**: ขาดข้อมูล Insight เกี่ยวกับลูกค้าและยอดขายย้อนหลัง

## Solution
ออกแบบระบบ UX/UI ที่เน้น Workflow การทำงานจริงลดความซับซ้อน
- **ใช้ UI แบบ Visual Floor Plan แบบ custom วางตามแผนผังร้านจริง ทำให้พนักงาน สะดวกและเห็นสถานะของโต๊ะทันทีจากสี**
- **รวมเมนูทั้งหมดไว้ใน Sidebar Navigation ไม่ต้องสลับหลายระบบ**
- **มี ปุ่ม CTA ชัดเจน (สร้างออเดอร์ใหม่ / เปลี่ยนพื้นที่) ลดการคลิก**

## Key Features & Functions
- **จัดการร้าน**: เช่น จัดการพนักงาน เมนูและโปรโมชั่น ตั้งค่าระบบ
- **จัดการ POS**: เช่น **จัดการโต๊ะ** ช่วยดูแผนผังโต๊ะ + เปิด/ปิดออเดอร์ , **จัดการออเดอร์** ติดตามรายการคำสั่งซื้อ และดูประวัติการขาย จัดการเงิน
- **จัดการสต๊อก**: เช่น ตรวจสอบวัตถุดิบ เชื่อมวัตถุดิบกับเมนู เพื่อคำนวณปริมาณตามสูตรแต่ละเมนู

## Outcome
- **บริการเร็วขึ้น**: พนักงานไม่ต้องเดินเช็กโต๊ะ แค่ดูจอก็รู้สถานะ ทั้งรายการออเดอร์ ยอดบิล สั่งอาหารเพิ่ม
- **ลดข้อผิดพลาด**: Color code ทำให้ไม่สับสนระหว่างโต๊ะที่จอง หรือโต๊ะที่มีลูกค้าแล้ว
- **โปร่งใสด้านการเงิน**: การปิดบิลชัดเจน audit ง่ายขึ้น
- **ข้อมูลเชิงลึก**: เจ้าของกิจการ/ผู้จัดการสามารถดูยอดขายย้อนหลัง และเมนูยอดฮิตที่ลูกค้าสั่งบ่อยที่สุด
