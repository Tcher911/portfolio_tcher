## Overview
เว็บแอปภายในรองรับ responsive เพื่อรวมศูนย์งานนำเข้า-ส่งออกเครื่องมือแพทย์ จัดการข้อมูล/ผู้ใช้, ยืนยันตัวตน, RBAC, แบบฟอร์มที่มี validation ลด user error, ความปลอดภัยCSP, และพร้อมใช้งาน Production ภายในเวลาพัฒนาเร่งด่วน 5 วัน

## Problem
- **ข้อมูลกระจายหลายแหล่ง**: ตามหาอุปกรณ์ ติดตามสถานะไม่ได้ และไม่รู้ ใครเป็นผู้นำเข้า-ส่งออก ในแต่ละเคส
- **ขั้นตอนทำงานแมนนวล**: ผิดพลาดจากผู้ใช้ ระบุข้อมูลผิดบ่อย, ตรวจย้อนกลับยาก
- **เวลาเครื่องมือหาย**: ไม่สามารถรู้ได้ว่าเครื่องมือหายไปกี่ชิ้น ใครเป็นคนนำเข้า เพราะ คนที่ส่งออก-นำเข้าไม่ใช่คนเดียวกัน

## Solution 
**ออกแบบระบบ Internal Tool**: ให้พยาบาลสามารถนำเข้า-ส่งออกเครื่องมือ และติดตามผู้นำเข้า-ส่งออก สถานะเครื่องมือได้อย่างสะดวก พร้อมหน้าตา UI ที่ใช้งานง่าย เพราะพยาบาลที่ใช้งานช่วงอายุแตกต่างกันมาก ตั้งแต่อายุ 23-55 ปี

## Key Features & Functions
- **Authentication & RBAC**: Sign-in/Sign-out จัดการ session ด้วย Supabase Auth รองรับ Role-based Access Control
- **Case & Asset Management**: สร้าง/ค้นหา/อัปเดต การนำเข้า-ส่งออกเครื่องมือ และรองรับการอัพเดทข้อมุล ในกรณีที่เครื่องมือหายและหาเจอในภายหลัง สามารถดูประวัติการนำเข้า-ส่งออกได้ทุกรายการ ฟิลด์ทุกจุดถูกคุมด้วย Zod schema เพื่อลดข้อมูลตกหล่น/ผิดชนิด
- **Lists, Filters & Detail Views**: รายละเอียด รวมข้อมูล ฟอร์มแก้ไข และเอกสารแนบในจุดเดียว เพื่อลดการสลับหน้าจอ
- **Forms & Validation UX**: มีการ validation ทุกที่ ลด drift และ user error
- **Security & Compliance**: Content Security Policy (CSP) พร้อม nonce, Security headers, การจัดการ Env/Secret ที่ปลอดภัย
- **Performance & Reliability**: Dynamic import / Lazy load / Image optimization ลด TTI และ bandwidth โครงสร้างโค้ด TypeScript + ESLint/Prettier เพื่อความเสถียรและดูแลรักษาง่าย
- **Admin Console**: จัดการผู้ใช้/บทบาท (เพิ่ม/ปิด/ปรับสิทธิ์) ดูภาพรวมระบบ

## Outcome
- **พร้อม Production**: ภายในกรอบเวลา 5 วันจาก timeline 7 วัน ระบบทำงานเสถียร ปลอดภัย และรองรับงานจริง
- **ไม่มี User Error**: ระบบตรวจสอบข้อมูลก่อนจะส่งฟอร์มออกไป ว่าตรงกับสต๊อกแ จำนวนคงเหลือของแต่ละรายการที่ส่งออก หรือประวัติการส่งออก-นำเข้าหรือไม่
- **รู้สถานะ-รู้ผู้รับผิดชอบ**: ของเครื่องมือทุกชิ้น—แก้ปัญหาของหาย/ไม่รู้ตัวรับผิดชอบ
- **ทำงานเร็วขึ้น**: ลดข้อผิดพลาดจากผู้ใช้/ข้อมูล ด้วยสคีมาเดียวและ UI ที่ออกแบบสำหรับงานจริง
- **เห็นภาพรวม**: จากแดชบอร์ด รู้จำนวนคงเหลือ จำนวนการส่งออก-นำเข้า เครื่องมือที่สูญหาย
