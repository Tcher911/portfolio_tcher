## Overview
โครงการนี้คือ Thailand Digital Arrival Card ของสำนักงานตรวจคนเข้าเมือง (Immigration Bureau) สำหรับผู้เดินทางต่างชาติที่เข้าประเทศไทย รองรับหลายภาษา

## Problem
- **ความล่าช้า**: ต้องตรวจสอบจำนวนมาก ทำให้เกิดความล่าช้า

## Solution
- **Traveler Management รองรับการเพิ่มผู้เดินทางหลายคนใน session เดียว กรอกได้รวดเร็วขึ้นสำหรับครอบครัว/กลุ่ม**
- **รองรับหลายภาษา ลดอุปสรรคการสื่อสาร**
- **ทำ UI แบบ Stepper 3 ขั้นตอน (Personal → Trip & Accommodation → Health Declaration) ลดภาระการกรอกครั้งเดียวเยอะๆ**

## Key Features & Functions
- **Landing Page**: ปุ่ม CTA 2 แบบ (Arrival Card ใหม่ / Update Arrival Card), dropdown เลือกภาษา, ข้อความแจ้งเตือนตามกฎหมาย
- **Stepper Form**: แบ่ง 3 ขั้นตอน → 1.Personal Information, 2.Trip & Accommodation, 3.Health Declaration
- **Traveler Management**: รองรับการเพิ่ม/เลือกผู้เดินทาง, ได้หลายคนใน session เดียว

## Outcome
- **ลดเวลา**: ที่ใช้ในการตรวจคนเข้าเมือง เพราะข้อมูลกรอกล่วงหน้า ทำให้เจ้าหน้าที่ตรวจได้เร็วขึ้น
- **ลดข้อผิดพลาด**: ระบบตรวจฟิลด์และบังคับรูปแบบอัตโนมัติ
- **ผู้ใช้มั่นใจ**: เพราะมีหลายภาษา ช่องทางช่วยเหลือ Error feedback ชัดเจน
- **ใช้งานสะดวกขึ้น**: สำหรับครอบครัว/กลุ่มทัวร์ด้วยฟีเจอร์ Add New Traveler รองรับการเพิ่มผู้ใช้หลายคน
- **สอดคล้องกฎหมายคนเข้าเมือง**: ช่วยให้การบริหารจัดการข้อมูลผู้เดินทางเป็นระบบมากขึ้น
